# 77 Car Service - Gestão Pro 🏁

Sistema de gestão completo para oficina mecânica premium.

## Módulos
- 📊 **Dashboard** - Visão geral em tempo real
- 👥 **CRM** - Gestão de clientes e histórico
- 🔧 **Ordens de Serviço** - Controle completo de O.S.
- 📦 **Estoque** - Controle de peças e insumos
- 📄 **Templates** - Modelos de documentos
- ⚙️ **Configurações** - Personalização do sistema

## Stack
- Next.js 14 (App Router)
- TailwindCSS
- Framer Motion
- Lucide Icons

## Instalação
```bash
npm install
npm run dev
```
